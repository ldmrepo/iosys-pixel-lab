**Step A — Add formatting helpers to AgentPanel.tsx:**

Add two helper functions near the top of the file (after existing helpers):

```typescript
function formatCost(usd: number): string {
  if (usd < 0.01) return '$0.00';
  return `$${usd.toFixed(2)}`;
}

function formatTokenCount(tokens: number): string {
  if (tokens >= 1_000_000) {
    return `${(tokens / 1_000_000).toFixed(1)}M`;
  }
  if (tokens >= 1_000) {
    return `${(tokens / 1_000).toFixed(1)}K`;
  }
  return String(tokens);
}
```

With conditional token display:
```jsx
{agent.tokenUsage && agent.tokenUsage.costUSD > 0 && (
  <div className="agent-card-tokens">
    <span className="agent-card-cost">
      {formatCost(agent.tokenUsage.costUSD)}
    </span>
    <span className="agent-card-token-sep">{'\u00B7'}</span>
    <span className="agent-card-token-count">
      {formatTokenCount(
        agent.tokenUsage.inputTokens + agent.tokenUsage.outputTokens
      )} tokens
    </span>
  </div>
)}
```

The condition `costUSD > 0` ensures the row only shows when there's meaningful data (not for freshly spawned agents with no turns yet).

The total token count combines input + output for the summary display (per user decision: `$0.42 . 12.3K tokens`). Cache tokens are tracked server-side for accurate cost but not shown separately in the compact card view.
