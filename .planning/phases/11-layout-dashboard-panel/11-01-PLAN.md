**App.tsx changes:**

1. Add panelCollapsed state with localStorage persistence (reuse the soundMuted pattern):
```typescript
const [panelCollapsed, setPanelCollapsed] = useState(() => {
  return localStorage.getItem('pixel-office-panel-collapsed') === 'true';
});

const handlePanelToggle = useCallback(() => {
  setPanelCollapsed(prev => {
    const next = !prev;
    localStorage.setItem('pixel-office-panel-collapsed', String(next));
    return next;
  });
}, []);
```

2. In the JSX, inside `.canvas-wrapper` div (which has `position: relative`), add the toggle button:
```tsx
<div className="canvas-wrapper">
  <OfficeCanvas ... />
  {tooltip && <Tooltip ... />}
  <button className="panel-toggle" onClick={handlePanelToggle} type="button">
    {panelCollapsed ? '\u25C0' : '\u25B6'}
  </button>
</div>
```
Note: Unicode arrows — left-pointing triangle when collapsed (click to expand = show panel), right-pointing triangle when expanded (click to collapse = hide panel). The button sits at the right edge of canvas-wrapper.

3. Pass collapsed state to AgentPanel:
```tsx
<AgentPanel
  agents={agentList}
  onAgentClick={handlePanelAgentClick}
  collapsed={panelCollapsed}
/>
```
